# 静脉输液操作流程

:::info 操作步骤
1. 洗手、戴口罩  
2. 核对医嘱，准备药品  
3. 吊瓶、排气，调节滴速  
4. 消毒穿刺，固定输液管  
5. 观察反应，记录数据

**注意防止空气栓塞与过敏**
:::

## 详细说明

### 1. 准备工作
- 洗手：使用七步洗手法
- 戴口罩：确保口罩完全覆盖口鼻
- 准备用物：输液器、消毒液、棉签等

### 2. 核对医嘱
- 核对患者信息
- 核对药品名称、剂量、浓度
- 检查药品有效期

### 3. 输液操作
- 检查输液器完整性
- 排气至输液管末端
- 调节滴速（成人一般40-60滴/分）

### 4. 穿刺固定
- 选择合适静脉
- 消毒穿刺部位
- 穿刺成功后固定

### 5. 观察记录
- 观察患者反应
- 记录输液时间、滴速
- 定期巡视

## 注意事项

1. 严格执行无菌操作
2. 防止空气栓塞
3. 注意观察过敏反应
4. 保持输液通畅
5. 及时处理异常情况
